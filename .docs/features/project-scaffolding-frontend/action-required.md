# Action Required: Project Scaffolding — Frontend

## Before Implementation

- [ ] **Install Node.js 20+** — Required for npm and Angular CLI. Download from https://nodejs.org
- [ ] **Install Angular CLI 21** — Run `npm install -g @angular/cli@21` to get the `ng` command

## After Implementation

- [ ] **Verify local dev server starts** — Run `npm start` in `./client` and confirm the app loads at http://localhost:4200 in a browser.
