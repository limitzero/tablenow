# Action Required: Project Scaffolding — Backend

## Before Implementation

- [ ] **Install .NET 10 SDK** — Required to run `dotnet build` and `dotnet ef`. Download from https://dot.net
- [ ] **Install EF Core CLI tools** — Run `dotnet tool install --global dotnet-ef` to enable migrations commands

## After Implementation

- [ ] **Create `appsettings.Development.json`** — Copy the template from `appsettings.json` and fill in a real JWT secret and local SQLite connection string. This file is git-ignored and must be created manually on each dev machine.
